Production rule

              
출시 기준

              

                
키 보안AI API 키는 브라우저가 아니라 Worker 환경변수에만 저장합니다.

                
모델 입력영상은 브라우저에서 프레임을 추출해 멀티모달 모델에 전달합니다.

                
의료 표현질병 확정 대신 위험 신호, 상담 필요도, 관찰 권장 행동만 안내합니다.

                
확장 앱가족 사진합성, 캐리커처, 게임은 별도 모듈과 Worker 엔드포인트로 추가합니다.

운영 상태베타 서비스이며 실제 병원 예약/보험 청구는 제공하지 않습니다.
