앱으로 돌아가기

개인정보 처리방침

시행일: 2026-05-05

1. 베타 서비스 저장 방식

현재 웹 베타는 서버 계정 없이 동작하며, 반려견 프로필, 건강기록, 영상 분석 이력, 커뮤니티 베타 게시글은 사용자의 브라우저 로컬 저장소에만 저장됩니다.

2. 수집 항목

사용자가 직접 입력한 반려견 이름, 품종, 나이, 체중, 알러지/기저질환 메모, 건강기록, 보호자 메모, 영상 URL, 음식 검색어, 게시글 내용을 저장할 수 있습니다.

3. 서버 전송

현재 버전은 입력 데이터를 별도 서버로 전송하지 않습니다. 외부 지도 버튼을 누르면 검색어가 네이버지도 또는 카카오맵으로 전달될 수 있습니다.

4. 데이터 삭제

앱의 요금제/내정보 화면에서 로컬 데이터 삭제를 실행하면 이 브라우저에 저장된 데이터가 삭제됩니다. 브라우저 설정에서 사이트 데이터 삭제를 실행해도 동일하게 제거됩니다.

5. 정식 출시 전 변경 예정

계정, 서버 저장, 결제, 병원 연계, 푸시 알림이 추가되면 개인정보 처리방침을 정식 서비스 기준으로 재작성하고 동의를 다시 받을 예정입니다.
